What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the structure of a web page as a tree of objects that can be manipulated with code.

How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById('elementId') , document.querySelector('.className'), document.getElementsByClassName(elementClass) etc.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element to manage events for all its descendants. It's useful for handling events efficiently, especially in dynamically changing content.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes using the DOM, you can use methods like setAttribute('attribute', 'value') to set a new attribute or change an existing one, and getAttribute('attribute') to retrieve the value of an attribute. For styles, you can access and modify them by directly changing properties of the element's style attribute. For example, element.style.color = 'red' would change the text color to red.
